# Modern Webapp 
By Daniel Bernstein
A  webapplication startup kit  webapp that is
* bundled
* live updatable
* setup for continuous unit and browser tests
* supports direct calls to aws via javascript api.
* supports openauth as well as local login
* supports anonymous access.
